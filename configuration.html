Fensterchef

The fensterchef configuration file format is very simple and easy to understand.

It is located at $XDG_CONFIG_HOME/fensterchef/config by default which itself defaults to .config/fensterchef/config.

Settings can be set through simple key value syntax. For example: border color #440088.

When a line starts with a hash # the line is ignored. Lines can be joined using a backslash \. This character must be the first non space character on a line which makes this line continue the previous line. 

Mod4+g
  \ show message
      \ "Hello, I am an example message"

Setting

There are a few settings to customize behavior and style of fensterchef.
auto equalize true or false

Make frames equalize their size when splitting/unsplitting.

auto fill void true or false

Fill empty frames with stashed frames automatically.

auto remove true or false

Remove any frames automatically even if gaps can be filled.

auto remove void true or false

Remove empty frames automatically.

auto split true or false

When a window is shown, split the current frame to make room to show it.

background color

Set the background color of fensterchef windows.

border color active color

Set the border color of active windows.

border color focus color

Set the border color of selected windows.

border color color

Set the border color of inactive and not selected windows.

border size number

Set the size of the border of all windows.

cursor horizontal cursor_name

Set the cursor shown when resizing horizontally.

cursor moving cursor_name

Set the cursor shown when moving a window.

cursor root cursor_name

Set the cursor shown on the root window.

cursor sizing cursor_name

Set the cursor shown when sizing a corner of a window.

cursor vertical cursor_name

Set the cursor shown when resizing vertically.

first window number number

Set the number the first window should receive.

font name

Set the font used for fensterchef windows.

foreground color

Set the text color of fensterchef windows.

gaps inner 1, 2 or 4 numbers

Set the gaps between frames and their inner windows.

gaps outer 1, 2 or 4 numbers

Set the gaps between frames and the monitor edges.

modifiers ignore modifiers

Set the modifiers to ignore for all bindings.

For example after setting this to Shift, then pressing Shift, will not mess with any bindings.

notification duration number

Set the duration the notification should be visible for.

overlap number

Set the percentage a window should be counted overlapping a monitor.

Set this to 100% to only allow this for fullscreen windows.

Set it to 101% to disable this feature.

text padding number

The insets of the text in fensterchef windows.

Action

Actions can be run directly in the configuration file or on a button/key binding. Actions may act on a window and if no window is directly specified, the selected window is used. select can be used to select a window.
assign number

Assign a number to the currently focused frame.

assign window number

Assign a number to the currently selected window.

center window

Center the selected window onto the monitor it is on.

center window to pattern

Center the selected window to the monitor matching the pattern.

If no monitor matches, nothing happens.

close window number?

Close the selected window or the window with given number.

dump layout file_name

Dump all frames and windows to a file.

exchange down

Exchange the current frame with the frame below it.

exchange left

Exchange the current frame with the frame left of it.

exchange right

Exchange the current frame with the frame right of it.

exchange up

Exchange the current frame with the frame above it.

focus child number?

Focus the child frame or the ith child frame.

focus down

Focus the frame below the current frame.

focus number

Focus the frame with given number.

focus leaf

Focus the deepest child of the current frame.

Same as focus child -1.

focus left

Focus the frame on the left of the current frame.

focus parent number?

Focus the 'nth parent frame.

focus right

Focus the frame on the right of the current frame.

focus root pattern?

Focus the most upper parent frame.

Same as focus parent -1.

focus up

Focus the frame on the above the current frame.

focus window number?

Focus the window with given number.

hint split horizontally

Hint that the current frame should split horizontally.

hint split vertically

Hint that the current frame should split vertically.

initiate move

Initiate a moving of a window with the mouse.

initiate resize

Initiate a resizing of a window with the mouse.

merge default

Merge the default configuration with the current configuration.

merge default bindings

Merge the default bindings into the current configuration.

merge default button bindings

Merge the default button bindings into the current configuration.

merge default cursor

Merge the default cursors into the current configuration.

merge default font

Merge the default font into current configuration.

merge default key bindings

Merge the default key bindings into current configuration.

minimize window number?

Minimize the focus window or the window with given number.

move down

Move the current frame down.

move left

Move the current frame to the left.

move right

Move the current frame to the right.

move up

Move the current frame up.

move window by x y

Move the selected window by given coordinates.

move window to x y

Move the selected window to given coordinates.

nop

Does nothing. Use this to grab keys/buttons or create relations that do

nothing.

pop stash

Pop the last stashed frame into the current frame.

quit

Quit fensterchef.

reload configuration

Clear the configuration and load all again from the configuration file.

remove number?

Remove the curent frame or the frame with given number.

resize window by x y

Resize the selected window by given coordinates.

resize window to x y

Resize the selected window to given coordinates.

run shell_program

Run a shell program.

The shell used is located at /bin/sh.

select focus

Select the selected window for further actions.

select pressed

Select the pressed window for further actions. This works only if it is used in a button binding.

select window number

Selet the window with given number for further actions.

set floating

Set the selected window to floating.

set fullscreen

Set the selected window to fullscreen.

set tiling

Set the selected window to tiling.

show list

Show/hide the interactive window list.

show message string

Show a notification window with given message.

show next window number?

Show the next window or the ith next window.

show previous window number?

Show the previous window or the ith previous window.

show run shell_program

Run a shell program and show its output as notification.

show window number?

Show the selected window or the window with given number.

split horizontally

Split the current frame horizontally.

split left horizontally

Split the current frame horizontally but swap the positions of the split frames.

split left vertically

Split the current frame vertically but swap the positions of the split frames.

split vertically

Split the current frame vertically.

toggle focus

Toggle the focus from the tiling layer to the floating layer.

toggle fullscreen

Toggle the mode of a window from X to fullscreen and vise versa.

toggle tiling

Toggle the mode of a window from X to tiling and vise versa.

Special

alias alias_name = string

Create an alias. Whenever a word (unquoted string) is used, it is attempted to resolve it to an alias. The exception is in alias un-/definitions.

unalias alias_name

Remove a previously declared alias

call group_name

Run all commands within the specified group.

group group_name commands

The group command creates a group of commands. Use call to run the commands within a group.

ungroup group_name

The ungroup command removes all bindings and relations a specific group created.

release? transparent? modifiers + button|key symbol commands

This creates a binding that triggers when the specified combination of modifiers and button/key symbol is pressed/released.

To get more information about the modifiers configured on your system, use xmodmap -pm. This shows the name of the modifier and then the physical keys that trigger these modifiers.

Note that fensterchef expects the modifiers to start with a capital letter. transparent can be used for button bindings to also send the button event to the pressed window. release can be used to trigger the bind when releasing the key/button.

Note that the ignored modifiers can be used to control which additional physically pressed modifiers should not change the interpretation of a binding.

Commands are separated by a comma: 

# Also trigger all bindings if Lock, Mod2 or Control is down
modifiers ignore Lock+Mod2+Control

# Show a simple message
Mod1+t show message Hello!

# Split a frame and focus the right frame
Mod1+Shift+t split horizontally, focus right

# Focus a window when pressed
transparent LeftButton focus window

Or put into brackets: 

# Hide the current window and focus/show the window number 5
Mod4+g (
    hide window
    select window 5, focus window
)

relate string commands The string refers to the instance name (resource name) and class name. You can use xprop to click on a window and read off the WM_CLASS property.

After the string comes the commands that should run when a window with this instance name and class name is created.

For example: 

# Set all firefox windows to floating and focus them when they appear
relate firefox set floating, focus window
unbind release? MODIFIERS + button|key_symbol

The unbind command can be used to remove a previously bound key or button.

unrelate

Remove the relation that is currently running (use within a relation).

unrelate all

Remove all window relations.

unrelate match|pattern string

Remove a window relation by matching given string or removing the relation defined with given exact pattern.

source file

The source command can source external files. This allows to split a configuration into multiple parts.

Example

Initial layout

relate "1,st-256color" assign window 901, focus window
relate "2,st-256color" assign window 902, focus window
relate "3,st-256color" assign window 903, focus window

assign 901

split horizontally
focus right
assign 902

split horizontally
focus right
assign 903

# Use st -n X, this sets the instance name to X
# (a feature of suckless terminal)
run 'st -n 1'
run 'st -n 2'
run 'st -n 3'

Shift+l move right
Shift+k move up

Resize mode

group resize (
    h resize window by -1%   0
    j resize window by   0  1%
    k resize window by   0 -1%
    l resize window by  1%   0
    
    q ungroup resize
)

Control+Mod4+r call resize

Default configuration

# Fensterchef version 2.0 configuration options

## Default settings ##

# Automatically resize frames
auto equalize true

# Automatically fill empty frames created
auto fill void true

# Automatically remove a frame when its window is closed
auto remove false

# Automatically remove empty frames when they appear implicitly
auto remove void false

# Automatically split a frame when a window is shown
auto split false


# Set the font used for fensterchef windows
font monospace

# Set the text color
foreground #7fd0f1

# Set the background color of fensterchef windows
background #49494d

# Set the offset within text rendered in fensterchef windows
text padding 6


# Set the border color of active windows
border color active #939388

# Set the border color of focused windows
border color focus #7fd0f1

# Set the color of inactive/unfocused windows
border color #49494d

# Set the size of borders
border size 1


# Set the cursor used on the root window
cursor root left_ptr

# Set the cursor used for moving windows
cursor moving fleur

# Set the cursor used for sizing horizontally
cursor horizontal sb_h_double_arrow

# Set the cursor used for sizing vertically
cursor vertical sb_v_double_arrow

# Set the cursor used for sizing window corners
cursor sizing sizing


# Set the gaps between frames
gaps inner 0 0 0 0

# Set the gaps between frames and monitors
gaps outer 0 0 0 0 

# Set the duration a notification window should stay for
notification duration 2

# When to count a window as overlapping a a monitor
overlap 80%


# Set the ignored modifiers
modifiers ignore Lock + Mod2
